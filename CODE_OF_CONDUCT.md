# Code of Conduct

## Our Standard

OpenTrader is a solo-developer project. I do as much testing as possible, but I am one person
designing, building, and maintaining the entire platform. Please keep that in mind when engaging
here.

Everyone who participates in this project's discussions, issues, and pull requests is expected to:

- **Be constructive** — describe problems clearly and suggest improvements helpfully
- **Be patient** — there are no SLAs; features and fixes happen when they can
- **Be respectful** — treat others (and the maintainer) the way you'd want to be treated
- **Be specific** — vague reports and demands help no one

## Unacceptable Behavior

The following will result in posts being removed and, for repeated offenses, account blocking:

- Rude, dismissive, or condescending language toward the maintainer or other users
- Demanding immediate fixes, updates, or responses
- Entitlement — this software is provided as-is; no one owes you anything
- Harassment of any kind

## Reporting

If you see a post that violates this code of conduct, flag it via GitHub's built-in report tools.
